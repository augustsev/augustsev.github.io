  • Unity

    Unity adalah salah satu game engine yang paling populer dan serbaguna. Ini mendukung pengembangan game untuk berbagai platform, termasuk PC, konsol, mobile, dan VR/AR.

  • Mobile Game

    Game mobile adalah jenis permainan video yang dirancang khusus untuk dimainkan di perangkat seluler, seperti ponsel pintar (smartphone) dan tablet.

  • GameMaker Studio

    GameMaker Studio adalah platform pengembangan game yang baik untuk pemula. Ini memiliki antarmuka yang ramah pemula dan menggunakan bahasa pemrograman scripting bernama GML (GameMaker Language).

  • Unreal Engine

    Ini memungkinkan penggunaan Blueprint (sistem visual scripting) atau C++ untuk pengembangan game.
